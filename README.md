# pv-mind-cockpit
PV_Mind Cockpit by GridMind EPC — cross-platform solar PV + BESS design, simulation, yield, CAPEX, BOM, SLD, SCADA, and reporting platform.

## SLD Preview

This repository includes a dynamic SLD preview page at:

- `/home/runner/work/pv-mind-cockpit/pv-mind-cockpit/index.html`

It supports:

- PV-only topology
- PV + BESS topology
- Dynamic display of:
  - PV DC Capacity
  - AC Capacity
  - Number of Inverters
  - Transformer Voltage
  - BESS MW / MWh
  - Export Limit
  - Power Factor
  - Grid Energy
